# V's Byte Corruptor

# Description
This app helps you manipulate the bytes of any file the way you like.

# Features
- Incrementing
- Randomizing
- Replacing
- Swapping
---
![ui](./gallery/ui.png)
---
# Usage
1. select a file
2. select where you want to save it
3. pick an algorithm (Incrementer, Randomizer, Replacer)
4. fill the boxes
5. run it
